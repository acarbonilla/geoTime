📱 Usage Instructions
  1. Search & Filter: Use the search bar to find specific requests, or use date/status filters
  2. Individual Reports: Click the PDF button in the Actions column of any request row
  3. Sample Report: Use the "Sample Report" button in the header to download the first visible request
  4. Bulk Download: Click "Bulk Download" to get all visible requests as a ZIP file
  5. Summary Report: Use "Summary Report" for statistical overview and analysis
⚠️ Important Notes
  • PDF generation requires modern browser support
  • Large bulk downloads may take time to process
  • Ensure sufficient memory for multiple PDF generation
  • Check browser download settings if files don't download
  • All reports include generation timestamps for record keeping
